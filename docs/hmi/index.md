# 🖥️ Panduan HMI & Panel Operator

Dokumentasi teknis mengenai penggunaan Human Machine Interface (HMI) pada sistem otomasi.

Panduan meliputi:

- Jenis-jenis HMI
- Desain tampilan (UI)
- Komunikasi HMI–PLC
- Alarm & Monitoring
- Setting port (RS485/Ethernet)
